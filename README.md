# myGulpFlow

## Installation

First of all, install the dependencies to run this boilerplate.

- [NodeJS](http://nodejs.org/)
- [GulpJS](http://gulpjs.com/)

```sh
# install gulp globally
$ npm install -g gulp

# install dependencies
$ npm install

```

## Tasks

```sh
# start project (watch files)
$ npm run start

# build project
$ npm run build
```
